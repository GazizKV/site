#
 Разработать сайт визитку (свое резюме)
 Дизайн на ваше усмотрение, обязательно должна быть ваша фотография, перечень мест учебы и работы,
информация об имеющихся знаниях/навыках, ваши контакты.
##
 Обязательные требования
###
 Common Skills
-
 Использовать `Git`,
 разбив на адекватное количество законченных по смыслу коммитов -
 Разместить результат на `GitHub
 Pages`
###
 CSS/HTML
-
 Использовать `Grid
 Layout`
-
 Использовать `Flexbox`
-
 Никаких `CSS`-фреймворков
-
 Использовать `SVG`
 иконки ####
 JS
-
 Реализовать редактирования/удаление перечня мест работы использую `Promise`,
`Async/await`,
 запрашивая с помощью `Fetch`
 какой-либо исходный `json`-файл
 при открытии сайта, изменения пользователя сохранять в `LocalStorage`
 и подмешивать их при перезапуске ##
 Будет плюсом (в порядке приоритета)
-
 Доработать редактирования/удаление перечня мест работы добавив имитацию бэкенда на
[JSON-Server](https://github.com/typicode/json-server),
 либо на любом его аналоге или на своем бэкенде, переместив туда все `CRUD`-операции
-
 Применить `TypeScript`,
 описав типы, а не добавлять просто везде `any`
-
 Применить `React`
-
 Применить `Mobx`
-
 Добавить сборку на `Webpack
 4`
 с транспиляцией `TypeScript`,
 минификацией результирующего `CSS/JS`
-
 Добавить полученную на предыдущем этапе сборку проекта в `GitHub
 CI (GitHub Actions)`
-
 Реализовать вариант Темной темы для сайта -
 Реализовать механизм разрушения элементов на страницы аля игра `Arkanoid`
 (подобная пасхалка есть на сайте `Yota`
 когда он не работает)
